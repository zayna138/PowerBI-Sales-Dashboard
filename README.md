# Power BI Sales Dashboard

## Project Overview
This project is an interactive sales dashboard created using Microsoft Power BI to analyze sales performance and generate business insights.

## Features
- Total sales analysis
- Product-wise sales tracking
- Profit analysis
- Monthly sales trends
- Interactive charts and visualizations

## Technologies Used
- Microsoft Power BI
- Excel Dataset

## Dashboard Components
- Card Visual for Total Sales
- Bar Chart for Product Sales
- Pie Chart for Profit Analysis
- Line Chart for Monthly Sales Trends

## Output
The dashboard provides clear visual insights into business performance using interactive reports and charts.
